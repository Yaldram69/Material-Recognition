# Material Recognition

A real-time **material recognition** project that classifies the material/surface type from a webcam feed using a fine-tuned **ResNet50** model.

---

## Features
- Real-time webcam inference
- ResNet50-based material classifier
- Predicts material category and overlays label on live video
- Runs on CPU or GPU (CUDA if available)

---

## Categories
brick, carpet, ceramic, fabric, foliage, food, glass, hair, leather, metal, mirror, other, painted, paper, plastic, polishedstone, skin, sky, stone, tile, wallpaper, water, wood
